# django-app-test
#### Данный проект создан для тестирования Django-приложений

## Инструкция по использованию
#### 1. Склонируйте репозиторий
```shell
git clone https://github.com/MagIlyasDOMA/django-app-test.git
```

#### 2. Перейдите в директорию репозитория
```shell
cd django-app-test
```

#### 3. Запустите setup.py
```shell
python setup.py
```

#### 4. Удалите папку django-app-test, если она не удалилась
```shell
cd ..

# В командной строке (Windows)
rd /s /q django-app-test

# В Powershell
Remove-Item -Path "django-app-test" -Recurse -Force

# В bash (Unix, Linux, Mac OS)
rm -rf django-app-test
```
